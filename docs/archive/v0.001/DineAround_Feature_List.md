
# 🍽️ DineAround – Complete Feature List

This document outlines all the current features for the DineAround app, categorized by type and tiered into Basic, Intermediate, and Premium levels.

---

## 🔍 Restaurant Discovery
- [ ] Detect current GPS location
- [ ] Show nearby restaurants using Google Places or Yelp API
- [ ] Display restaurant name, address, rating, photo
- [ ] Manually search for restaurants

---

## 📓 Restaurant Logbook (Visited)
- [ ] Add restaurant to personal log
- [ ] Date/time of visit
- [ ] Add personal notes
- [ ] Custom rating system
- [ ] Add tags (e.g., halal, casual, romantic)
- [ ] Upload photos (food, place, menu)
- [ ] Auto-fill restaurant info via API
- [ ] Timeline or map of visited places
- [ ] Filter visits by city, tags, or date
- [ ] Export/share log entry (image or private link)

---

## 📷 Menu Capture + Digitization
- [ ] Take or upload photo of menu
- [ ] OCR extracts dishes + prices
- [ ] Manually edit parsed menu
- [ ] Save structured digital menu under restaurant
- [ ] Auto-organize menu into categories (e.g., Appetizers, Mains)
- [ ] Translate menu to English (optional)

---

## 📌 Wishlist / “Want to Visit”
- [ ] Add restaurant manually or via search
- [ ] Tag with trip or interest (e.g., Tokyo 2025, Michelin)
- [ ] Notes per wishlist entry
- [ ] Map view of wishlisted places
- [ ] Filter wishlist by location or tag

---

## 📍 Location-Based Smart Alerts
- [ ] Notify when user is near a wishlist restaurant
- [ ] Map highlights nearby wishlist locations
- [ ] Local push notifications (background geofence)
- [ ] Toggle alert feature on/off

---

## 🌐 External Review Integration
- [ ] Display Google Reviews
- [ ] Display Yelp Reviews
- [ ] Side-by-side comparison
- [ ] Pull in review snippets and average score

---

## 🧠 Smart Features
- [ ] Full-text search across notes, tags, places
- [ ] Suggest restaurants based on past logs
- [ ] Monthly summary of dining history
- [ ] Track favorite cuisines or most visited cities
- [ ] Auto-tagging via API data

---

## 🔐 Privacy + Sync
- [ ] App is private by default
- [ ] Optional cloud sync (Firebase/Supabase)
- [ ] Email or Apple Sign-In for multi-device sync
- [ ] Manual export/backup of data
- [ ] Share links only when user initiates

---

## 🪜 Feature Tiers

### 🟢 Basic (MVP)
- Manual logging
- Local data storage
- GPS detection
- Wishlist creation
- Share logs manually

### 🟡 Intermediate
- OCR menu scanning
- Real restaurant lookup (Google/Yelp)
- Local notifications
- Basic map view

### 🔵 Premium
- Smart filtering/search
- Monthly reports
- Taste-based suggestions
- Cloud backup + sync
- Multi-device support
- Smart OCR enhancements
